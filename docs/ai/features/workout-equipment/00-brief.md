# Workout Equipment Selection — Brief

## Цель
В админ-панели при создании и редактировании тренировки администратор может указать требуемое оборудование.  
Выбор осуществляется через чек-боксы.  
Список оборудования берётся из `const equipmentItems`.

## Где
- Админ-панель
- Страница создания тренировки
- Страница редактирования тренировки

## Функционал

- Отображается список оборудования в виде чек-боксов.
- Источник данных — `const equipmentItems` (единый источник истины).
- Можно выбрать несколько позиций.
- При сохранении тренировки выбранное оборудование сохраняется в БД.
- При редактировании тренировки ранее выбранные позиции автоматически отмечены.
- Если оборудование не выбрано — поле сохраняется как пустой массив.
- Валидация: допускаются только значения, присутствующие в `equipmentItems`.
- Проверка прав на редактирование
