# Support Chat — Brief

## Цель
Пользователь может писать в поддержку.
Поддержка может отвечать.

## Где
- Пользователь: личный кабинет
- Поддержка: админ-панель

## Функционал
- Можно вставлять emoji
- realtime используя SSE
- Можно добавлять вложения, изображения, видео, 
- Telegram уведомление при новом сообщении пользователя через telegram бота.
- прочитанные сообщения имеют индикатор прочитано

## UI
- диалог как в telegram
- учитывать цвета для светлой и темной темы

## Definition of Done
- Пользователь видит свои диалоги
- Поддержка видит все диалоги
- Telegram уведомление приходит
- Нет утечки чужих диалогов
